# 🧠 Sentiment Analysis Dashboard

An interactive **Natural Language Processing (NLP)** dashboard that performs real-time sentiment and emotion analysis on user input text.

Built with **Streamlit**, this app not only shows sentiment scores but also explains *why* a certain sentiment was detected.

---

## 🚀 Features

- ✅ Real-time sentiment analysis using **VADER**
- 🎭 Emotion detection (joy, anger, fear, sadness, surprise, trust, disgust)
- 📊 Interactive **Emotion Radar Chart** (Plotly)
- 🔍 Explainable AI (word-level sentiment scoring)
- ✨ Intensity word highlighting (e.g., *very*, *extremely*, !!!)
- 📈 Additional insights using **TextBlob**
- 🌐 Live sharing using **Ngrok**

---

## 🛠️ Tech Stack

- Python 🐍  
- Streamlit  
- VADER Sentiment Analyzer  
- TextBlob  
- Plotly
